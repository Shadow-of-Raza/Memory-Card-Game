# Memory Card Game
A modern, responsive memory card game built with vanilla JavaScript, HTML5, and CSS3. Test your memory skills by matching pairs of playing cards in this beautifully designed game with smooth animations and immersive sound effects.

# Features
## Core Gameplay

* Card Matching: Flip cards to find matching pairs
* Multiple Difficulty Levels:
  * Easy (4×4 grid - 8 pairs)
  * Medium (6×6 grid - 18 pairs)
* Game Statistics: Track moves, time, and matches
* Win Celebration: Confetti animation with sound effects

## User Experience

* Smooth Animations: Card flip, match, and error animations
* Sound Effects:
  * Card flip sounds
  * Match celebration
  * Mismatch feedback
  * Win celebration
* Sound Control: Toggle sound on/off during gameplay
* Responsive Design: Works perfectly on desktop, tablet, and mobile

## Visual Design

* Professional Card Design: Standard playing card faces with proper suits and ranks
* Glassmorphism UI: Modern translucent elements with backdrop blur
* Gradient Background: Beautiful color transitions
* Responsive Grid: Adaptive card sizing for different screen sizes

## How to Play
1. Start Game: Click "New Game" to begin
2. Choose Difficulty: Select between 4×4 (Easy) or 6×6 (Medium) grid
3. Flip Cards: Click on cards to flip them over
4. Find Matches: Match two identical cards to make them disappear
5. Complete: Match all pairs to win the game
6. Track Progress: Monitor your moves and time in the dashboard

# Game Rules

* Click any card to flip it over
* Click a second card to try and find a match
* If cards match, they remain face up
* If cards don't match, they flip back after 1 second
* Game ends when all pairs are matched
* Try to complete the game with the fewest moves and fastest time

## Customization

The game is easily customizable:
* Add new difficulty levels by modifying grid sizes
* Replace card images in the faces/ directory
* Customize sounds by replacing audio files
* Modify colors and animations in CSS
